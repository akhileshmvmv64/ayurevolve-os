# Architecture

## Overview
AYUREVOLVE OS is a Next.js App Router application for enterprise operations, knowledge, analytics, and AI collaboration.

## Core layers
- Frontend: Next.js 16, React 19, TypeScript
- Styling: Tailwind CSS
- Auth: Supabase Auth with SSR cookies
- Data: Supabase Postgres
- Routing: App Router with protected dashboard routes

## Principles
- Secure by default
- Modular UI components
- Route-protected experiences
- Shared enterprise design system
